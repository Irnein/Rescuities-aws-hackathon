# Rescuties

Rescuties is a disaster-response project developed during the AWS Great Malaysia AI Hackathon. The system was designed to analyze social media content, extract disaster-related information, and support situational awareness for emergency response.

## Overview
The project explores how cloud and AI services can be used to process public information related to disasters and convert it into structured, useful outputs.

## Features
- Social media data processing
- Disaster-related text analysis
- Information extraction using Amazon Comprehend
- AI-assisted summarization using Amazon Bedrock
- Structured alert generation
- AWS-based processing workflow

## Tech Stack
- Python
- AWS Lambda
- Amazon Kinesis
- Amazon Bedrock
- Amazon Comprehend

## Project Structure
- `src/` - source code for processing scripts and Lambda logic
- `configs/` - configuration and JSON/YAML files
- `docs/` - project notes and supporting information
- `backups/` - backup templates or archived files
